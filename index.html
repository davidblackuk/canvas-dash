Features

  • 100% HTML5 Canvas based
  • Implemented in Typescript, also available in javascript
  • Open source
  • Use via a jQuery plugin
  • Animated and themable
  • Highly customizable
    • Colors
    • Fonts
    • Packaged as themes
  • Dials
    • 360 and 180 degree dials
    • Horizontal sliders
    • Scrolling marquee
